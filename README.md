# SQL Foundations

## Overview
This repository contains hands-on SQL assignments created as part of my data science learning journey. The projects focus on practical database operations and analytical queries using structured datasets.

## Topics Covered
- Database creation and table design
- Data insertion and modification
- Joins (INNER, LEFT)
- Aggregate functions (SUM, COUNT, AVG)
- Subqueries
- CASE statements for segmentation
- Date functions and filtering
- Data analysis using SQL

## Files Description
- **01_sakila_sql_queries.sql**  
  SQL queries executed on the Sakila sample database to practice joins, subqueries, aggregations, and reporting queries.

- **02_sales_analysis_sql.sql**  
  End-to-end SQL analysis using a simulated sales dataset including customers, products, and orders. Covers revenue analysis, customer segmentation, and time-based analysis.

## Tools Used
- MySQL
- SQL

## Purpose
This repository demonstrates my practical understanding of SQL for data analysis and database management.
